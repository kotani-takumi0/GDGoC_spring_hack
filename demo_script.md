## 第1幕: 問題提起（スライド + 口頭）

### 1-1. Hello Worldの崩壊

> 「Hello Worldって、初心者が最初にやることでした。
> 1行。書いて、動いて、理解できた。だから一歩目として成立していた。」

> 「今、初心者がAIにアプリを作らせると、数百行のコードが返ってくる。
> 動く。でも理解できない。
> 出力と理解の間に、初心者ほど超えられないギャップが生まれている。」

> 「Hello Worldは、もう一歩目として機能していない。」

### 1-2. 聴衆への問いかけ

> 「皆さんもこの1週間でAI使いましたよね？」
> 「そのAIが書いたコード、全部説明できますか？」
> 「もちろん全部説明できなくてもいい。
> でも、ここだけは理解しておきたいって思う箇所ありませんか？」
> 「じゃあそれ、いつ学ぶのか。」

### 1-3. プロダクト導入

> 「僕たちは、AIがコードを書いているまさにその時間に、理解する場所を作りました。」

→ 画面をプロダクトに切り替え

---

## 第2幕: プロダクト体験（デモ）

### 2-1. AIにコード生成を指示する

画面: Antigravity（IDE）上で「Todoアプリを作って」と入力

> 「今からAIにTodoアプリを作らせます。」

→ Gemini 3 がタスク分解を開始

### 2-2. 構造マップの骨格が出現する【見せ場①】

画面: 理解支援アプリに切り替え。構造マップに全ノードがグレーで一斉に出現。

> 「AIがまずタスクを分解しました。
> この5つのファイルがこれから作られます。
> まだ1行もコードは書かれていません。」

観客が見ているもの:
- 左パネル: ファイルツリー（App.jsx, TodoList.jsx, TodoItem.jsx, useTodos.js, storage.js）全てグレー
- 右パネル: ファイル間の依存関係がエッジで繋がっている
- 全てグレー = 「これから作られる」感

### 2-3. 事前学習: コードが書かれる前に全体像を掴む

画面: 事前学習の予想問題がポップアップ

> 「コードが書かれる前に、まず全体像を掴みます。」

問題例:
```
Q. 「Todoアプリでユーザーが『追加』ボタンを押したとき、
    最初に動くのはどのファイルだと思いますか？」
   [App.jsx]  [TodoList.jsx]  [useTodos.js]  [storage.js]
```

→ 正解すると構造マップ上でApp.jsx → TodoList → useTodosのエッジがアニメーションで光る

> 「コードを1行も見ていないのに、もうアプリの構造が頭に入り始めている。」

### 2-4. AIがコードを書き始める【見せ場②】

画面: 構造マップ上でApp.jsxノードがグレー点滅を開始

> 「AIがApp.jsxを書き始めました。」

→ AIの現在地パネルに思考ステップが流れる
```
🤖 AIの現在地
  App.jsx を書いています
  💭 「useStateでTodoの配列を管理します」
  ステップ: ██░░░░ 1/5
```

> 「AIが何を考えて、なぜそう書いているかが、リアルタイムで見えます。」

### 2-5. 実装完了 → 壊して予想【見せ場③】

画面: App.jsxノードがグレー点滅 → 白（静止）に変化。内部ノード（useState, handleAdd, return JSX）が展開。

> 「App.jsxが書き終わりました。中身が見えます。」

→ 壊して予想の問題が出る
```
🔧 App.jsx の理解チェック

  このコードから useState を取り除くと
  アプリはどうなると思いますか？

  [A] Todoを追加しても画面に表示されない
  [B] アプリがエラーで動かなくなる
  [C] 見た目は変わらないがリロードで消える
```

→ 予想を選択 → 結果表示（正解 or 解説）

> 「予想して、結果を知る。この繰り返しが一番記憶に残る学び方です。」

### 2-6. ノードが緑に変わる【見せ場④ = クライマックス】

画面: 予想が当たり、App.jsxのuseStateノードが白点滅 → 🟢緑に変化

→ アプリが画面に表示:

```
🎉 Hello, World.
```

> 「最初のノードが緑に変わった。
> これがAI時代の、新しいHello Worldです。」

（間を取る）

### 2-7. 並走の全体像を見せる（オプション・時間があれば）

画面: 構造マップ全体を引きで見せる。AIが次のファイル（TodoList.jsx）を書き始めてグレー点滅。App.jsxは緑。

> 「AIはまだ書いている。でも自分はもう、ここを理解している。
> 出力と理解が、もう一度並走している。」
